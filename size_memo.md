# サイズのメモ

### Githubの推奨サイズ

*  プロフィール画像
推奨サイズ : 500 × 500 ピクセル
ファイル制限 : 1MB 以下（PNG, JPG, GIF） 

* 横幅いっぱいのヘッダー・大型画像
	作成サイズ : 1700 × 850 ピクセル（2:1）
	表示サイズ : 横幅 850 ピクセル

* ロゴ・アイコン・バッジ

	作成サイズ : 200 × 200 ピクセル
	表示サイズ : 横幅 100 ピクセル（またはそれ以下）
	
	または、
	 実際の作成サイズ（画像ファイル自体）:　 縦横 80 × 80 〜 100 × 100 ピクセル
	READMEでの表示サイズ:** 縦横  40 × 40 ピクセル 前後



### Xの推奨サイズ

* 1枚投稿
　縦長 3 : 4 ( 900x1200 )
　正方形 1 : 1 ( 1200x1200 )
　横長 16 : 9 ( 1200x675 )

* 2枚投稿
	8 : 9 ( 1200x1350 )　x2枚

* 3枚投稿
	8 : 9 ( 1200x1350 )   x1枚
	16 : 9 ( 1200x675 ) 　x2枚

* 4枚投稿
	16 : 9 ( 1200x675 ) 　x4枚

* プロフ画像
	 1 : 1 ( 400x400 )

* ヘッダー画像
	3 : 1 ( 1500x500 )

### kaggleの推奨サイズ

* 推奨解像度 : 512 × 512 ピクセル（またはそれ以上の正方形画像）
最低解像度 : 256 × 256 ピクセル以上

### NOTEの推奨サイズ

* 記事の見出し画像 : 1280 × 670 px または 1280 × 670px
  
* クリエイターページ/マガジンヘッダー画像 : 1920×1006 px

* プロフィールアイコン : 330 x 330px

* メンバーシップ画像 : 1280 × 720 px または 1920×1080 px
